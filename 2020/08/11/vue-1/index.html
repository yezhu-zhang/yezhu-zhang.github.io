3. 模版语法-指令

vue提供的固有指令
v-text 模版语法-插值
v-html 模版语法-插值 - 原始html
v-show 条件判断
v-if 条件判断
v-else 条件判断
v-else-if 条件判断
v-for 列表循环
v-on 绑定事件
v-bind 绑定属性
v-model 双向绑定
v-slot 插槽
v-pre 跳过这个元素和它的子元素的编译过程
v-cloak 这个指令保持在元素上直到关联实例结束编译。
v-once 只渲染元素和组件一次

自定义指令
全局自定义指令+局部自定义指令
需要对普通 DOM 元素进行底层操作，这时候就会用到自定义指令

全局的自定义指令 – 一定是写在new Vue 之前
使用时 v-focus

Vue.directive('focus', {
  // 当被绑定的元素插入到 DOM 中时……
  inserted (el) {
    el.focus()
  }
})

局部自定义指令

const vm = new Vue({
  el: '#app',
  directives: { // 局部自定义指令
    focus: {
      inserted (el) {
        el.focus()
      }
    }
  }
})

其他的钩子函数

7. 表单的输入

v-model
checkbox 如果数据类型为 数组 —- 表示多选
如果是布尔类型，表示选中还是未选中

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>

  <div id="app">
    <input type="text" placeholder="用户名" v-model="username"/> <br/>
    <input type="password" placeholder="密码" v-model="password"/> <br/>
    <input type="email" placeholder="邮箱" v-model="email"/> <br/>
    <input type="radio" name="sex" value="1" v-model="sex"/>男 
    <input type="radio" name="sex" value="0" v-model="sex"/>女<br/>
    <select v-model="lesson">
      <option value="1">1阶段</option>
      <option value="2">2阶段</option>
      <option value="3">3阶段</option>
    </select><br/>
    <input type="checkbox" name="hobby" value="篮球" v-model="hobby"/>篮球
    <input type="checkbox" name="hobby" value="网球" v-model="hobby"/>网球
    <input type="checkbox" name="hobby" value="足球" v-model="hobby"/>足球
    <input type="checkbox" name="hobby" value="桌球" v-model="hobby"/>桌球<br/>
    <textarea v-model="note" placeholder="备注"></textarea>

    <input type="checkbox" v-model="flag">阅读相关的信息

    <button @click="getData">获取用户信息</button>
  </div>
</body>
<script src="dist/vue.js"></script>

  const vm = new Vue({
    el: '#app',

      username: '',

        console.log({
          username: this.username,

          lesson: this.lesson * 1 === 1 ? '1阶段' : this.lesson * 1 === 2 ? '2阶段' : '3阶段',
          hobby: this.hobby,
          note: this.note,
          flag: this.flag ? '已勾选用户协议' : '未勾选'

  })

</html>
